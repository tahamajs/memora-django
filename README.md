# memora-django

A production‑ready Django backend for local‑first note‑taking.  
Part of the **Memora** ecosystem, it provides:

- Full note CRUD with Markdown
- AI integration (OpenAI summarisation, tags, writing improvement)
- Git version control & GitHub sync
- Research notes, categories, tags
- JWT authentication
- File attachments (local / S3 / R2 / MinIO)
- Webhooks & task queue (Celery)
- SCIM 2.0, GDPR tools, audit logging
